# api-laravel-vue
Proyecto API laravel + vue2.js
Laravel v8.33.1 (PHP v7.4.8)
API with Laravel Passport

--------------------------------
        INSTALLATION
--------------------------------

1 - run > composer install
2 - run > npm install
3 - create db and config .env
4 - run > php artisan migrate:db seed
5 - php artisan passport:install
